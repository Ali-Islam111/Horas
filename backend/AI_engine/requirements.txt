# INSTALL ORDER — follow exactly to avoid conflicts:
#
#   Step 1: Install PyTorch with CUDA (run setup_cuda.bat, or manually):
#           pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
#
#   Step 2: Install CTranslate2 before faster-whisper (avoids binary conflict):
#           pip install ctranslate2>=4.0.0
#
#   Step 3: pip install -r requirements.txt
#
# VERSION CONFLICT NOTES:
#   - numpy: mediapipe requires numpy<2.0; pinned to <2.0 to prevent breakage
#   - tensorflow: 2.15.x is the last version that bundles keras 2 AND works
#     with tensorflow-hub. TF 2.16+ moved to keras 3 and breaks hub models.
#     deepface 0.0.93+ requires tf-keras separately — installed below.
#   - faster-whisper: requires ctranslate2>=4.0, which requires numpy<2.0
#     (already covered). Do NOT install openai-whisper alongside it.
#   - mediapipe: 0.10.14+ dropped support for Python 3.8. Requires numpy<2.0.
#   - ultralytics (YOLO11): requires torch>=2.0 — satisfied by Step 1 above.
#   - deepface: 0.0.93+ no longer bundles keras; needs tf-keras shim below.

# ── Audio: speech transcription ───────────────────────────────
